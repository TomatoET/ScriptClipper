# Repository Instructions

## Global Instructions

- 对飞书文档进行任何修改或编辑时，不要直接覆盖原文；所有改动都必须以蓝色文字颜色编辑，修改则加删除线的方式呈现。

## Git Workflow

- Do not modify `main` directly.
- All future code changes must be made on a feature branch.
- All future code changes must be reviewed and merged through a pull request.
- Do not commit secrets, tokens, cookies, account credentials, `.env` files, or machine-specific absolute paths.
- Do not commit generated artifacts such as `__pycache__/`, virtual environments, build outputs, logs, or temporary files.
- Before proposing a commit, check the working tree and summarize the exact files that will be included.
